# Task D18-003: Add referral program constants

## What to Build

Add referral program constants to `packages/shared/src/constants/platform.constants.ts`. These support the redesigned referral program (Pricing Gap Closure Spec B3-1 through B3-5).

**Add the following constants:**

### Referral Statuses Enum

```typescript
export const ReferralRedemptionStatus = {
  PENDING: 'PENDING',
  QUALIFIED: 'QUALIFIED',
  CREDITED: 'CREDITED',
  EXPIRED: 'EXPIRED',
} as const;

export type ReferralRedemptionStatus =
  (typeof ReferralRedemptionStatus)[keyof typeof ReferralRedemptionStatus];
```

- `PENDING` — code redeemed, waiting for referred user to complete one billing cycle
- `QUALIFIED` — referred user completed billing cycle, credit value calculated, awaiting application
- `CREDITED` — credit applied to referrer's invoice or banked
- `EXPIRED` — referral did not qualify (cap exceeded, referred never paid, etc.)

### Credit Application Targets Enum

```typescript
export const CreditApplicationTarget = {
  PRACTICE_INVOICE: 'PRACTICE_INVOICE',
  INDIVIDUAL_BANK: 'INDIVIDUAL_BANK',
} as const;

export type CreditApplicationTarget =
  (typeof CreditApplicationTarget)[keyof typeof CreditApplicationTarget];
```

- `PRACTICE_INVOICE` — credit applied as negative line item on practice's next Stripe invoice
- `INDIVIDUAL_BANK` — credit held for future individual subscription use

### Referral Program Numeric Constants

```typescript
export const REFERRAL_MAX_CREDITS_PER_YEAR = 3;
export const REFERRAL_CODE_LENGTH = 8;
export const REFERRAL_CREDIT_CHOICE_DEADLINE_DAYS = 7;
```

- Max 3 credits per referrer per anniversary year (replaces old 10/year cap)
- Referral codes are 8-character alphanumeric strings
- Clinic physicians have 7 days to choose credit application target; defaults to PRACTICE_INVOICE

### Referral Credit Value Rates (reference comment)

Add a comment block explaining credit value calculation:
```typescript
// --- Referral Credit Values ---

//   Early Bird:           $199.00
```

**Do NOT modify any existing constants.** Only add the new referral-related constants at the end of the file, grouped under a `// --- Referral Program ---` section header.
