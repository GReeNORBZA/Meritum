# Task SCR-001: Extract shared scraper utilities into scripts/lib/fee-navigator-utils.ts

## What to Build

Create `scripts/lib/fee-navigator-utils.ts` containing all utility functions currently duplicated between `scripts/scrape-fee-navigator.ts` and `scripts/enrich-hsc-data.ts`.

**Functions to extract:**

1. `sleep(ms: number): Promise<void>` — Promise-based delay.

2. `fetchWithRetry(url: string, options?: RequestInit, retries?: number): Promise<string>` — HTTP fetch with retry logic and exponential backoff for 429/503 responses. Keep the current implementation for now (SCR-002 will enhance it).

3. `decodeHtmlEntities(xml: string): string` — Extract `<content>` from XML response and decode HTML entities. Keep the current implementation for now (SCR-002 will enhance it).

4. `saveJson(outputDir: string, filename: string, data: unknown): void` — Write JSON to file with `JSON.stringify(data, null, 2)`. Create directory if needed.

5. `loadJson<T>(outputDir: string, filename: string): T | null` — Read and parse JSON file, return null if file doesn't exist.

6. `ensureDir(dir: string): void` — Create directory recursively if it doesn't exist.

**Shared constants to extract:**

```typescript
export const BASE_URL = 'https://apps.albertadoctors.org/fee-navigator';
export const DELAY_MS = 200;
export const MAX_RETRIES = 3;

export const HEADERS: Record<string, string> = {
  'User-Agent': 'Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36',
  Referer: `${BASE_URL}/hsc`,
  'X-Requested-With': 'XMLHttpRequest',
};
```

**Then update both scripts:**

1. In `scripts/scrape-fee-navigator.ts`: Remove the duplicated functions and constants. Add `import { sleep, fetchWithRetry, decodeHtmlEntities, saveJson, loadJson, ensureDir, BASE_URL, DELAY_MS, MAX_RETRIES, HEADERS } from './lib/fee-navigator-utils.js';` at the top. The `saveJson` calls in the scraper currently take just a filename and use a module-level `OUTPUT_DIR` — adapt the import usage so the scraper passes its `OUTPUT_DIR` to `saveJson(OUTPUT_DIR, filename, data)`.

2. In `scripts/enrich-hsc-data.ts`: Remove the duplicated functions and constants. Add the same import. The enrichment script has its own `DATA_DIR` — use `loadJson(DATA_DIR, 'hsc-codes.json')` etc.

**Important:** Do NOT change any function behavior in this task. Just extract and import. SCR-002 will enhance the functions.
