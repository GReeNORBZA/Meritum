### Fix 2: Cache JSON loading in `loadScrapedCode` (line ~141)

**Current:** Reads and parses the 2.4 MB `hsc-codes.json` file on every call (9 times for 9 sample codes).

**Fix:**
```typescript
let _scrapedCodesCache: Array<Record<string, unknown>> | null = null;

function loadScrapedCode(hscCode: string): Record<string, unknown> | null {
  if (!_scrapedCodesCache) {
    const filePath = path.join(OUTPUT_DIR, 'hsc-codes.json');
    if (!fs.existsSync(filePath)) return null;
    _scrapedCodesCache = JSON.parse(fs.readFileSync(filePath, 'utf-8'));
  }
  return _scrapedCodesCache!.find(
    (c: Record<string, unknown>) => c.hscCode === hscCode,
  ) ?? null;
}
```

### Fix 3: Clean up event listeners

#### In `probeForPopups` (line ~251):

The function adds a `page.on('dialog', ...)` listener each time it's called but never removes it. After exploring 9 pages, 9 dialog listeners accumulate.

**Fix:** Use a named handler and remove it at the end of the function:

```typescript
async function probeForPopups(page: Page): Promise<PopupContent[]> {
  const popups: PopupContent[] = [];

  const dialogHandler = async (dialog: import('playwright').Dialog) => {
    popups.push({
      triggerElement: 'browser-dialog',
      triggerText: '',
      dialogContent: dialog.message(),
      dialogTitle: dialog.type(),
    });
    await dialog.dismiss();
  };

  page.on('dialog', dialogHandler);

  try {
    // ... existing popup probing logic (unchanged) ...
  } finally {
    page.removeListener('dialog', dialogHandler);
  }

  return popups;
}
```

#### In `monitorNetworkRequests` (line ~751):

Same issue with `page.on('request', ...)`. The listener should be removed after monitoring is complete.

**Fix:** Use a named handler and remove it at the end:

```typescript
async function monitorNetworkRequests(
  page: Page,
  hscCode: string,
): Promise<string[]> {
  const ajaxUrls: string[] = [];

  const requestHandler = (request: import('playwright').Request) => {
    const url = request.url();
    if (url.includes('fee-navigator') && request.resourceType() !== 'image') {
      ajaxUrls.push(`${request.method()} ${url}`);
    }
  };

  page.on('request', requestHandler);

  try {
    // ... existing navigation and interaction logic (unchanged) ...
  } finally {
    page.removeListener('request', requestHandler);
  }

  return ajaxUrls;
}
```
