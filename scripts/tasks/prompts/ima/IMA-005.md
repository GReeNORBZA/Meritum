# Task IMA-005: Generate and apply database migration

## What to Build

Generate a Drizzle migration from the schema changes made in IMA-002 and IMA-003:
- New column: `users.secondary_email` (varchar 100, nullable)
- New table: `ima_amendments`
- New table: `ima_amendment_responses`
- New table: `breach_records`
- New table: `breach_affected_custodians`
- New table: `breach_updates`
- New table: `data_destruction_tracking`

Run `drizzle-kit generate` to produce the migration file. Do NOT edit the generated migration file.

Then apply the migration with `drizzle-kit migrate`.

## Prerequisites

IMA-002, IMA-003 (schema definitions must be complete and building)

## Run After Completion

```bash
cd apps/api && pnpm drizzle-kit generate 2>&1 && pnpm drizzle-kit migrate 2>&1
```
