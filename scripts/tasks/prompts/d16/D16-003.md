# Task D16-003: Drizzle schema for practice_memberships table with billing_mode column

## What to Build

Add the `practiceMemberships` table to `packages/shared/src/schemas/db/platform.schema.ts`. Per Pricing Gap Closure Spec B1-2.

**Table definition:**

```typescript
export const practiceMemberships = pgTable(
  'practice_memberships',
  {
    membershipId: uuid('membership_id').primaryKey().defaultRandom(),
    practiceId: uuid('practice_id')
      .notNull()
      .references(() => practices.practiceId),
    physicianUserId: uuid('physician_user_id')
      .notNull()
      .references(() => users.userId),
    billingMode: varchar('billing_mode', { length: 30 })
      .notNull()
      .default('PRACTICE_CONSOLIDATED'),
    joinedAt: timestamp('joined_at', { withTimezone: true })
      .notNull()
      .defaultNow(),
    removedAt: timestamp('removed_at', { withTimezone: true }),
    removalEffectiveAt: timestamp('removal_effective_at', {
      withTimezone: true,
    }),
    isActive: boolean('is_active').notNull().default(true),
    createdAt: timestamp('created_at', { withTimezone: true })
      .notNull()
      .defaultNow(),
  },
  (table) => [
    uniqueIndex('practice_memberships_practice_physician_active_idx')
      .on(table.practiceId, table.physicianUserId)
      .where(sql`${table.isActive} = true`),
    uniqueIndex('practice_memberships_physician_active_idx')
      .on(table.physicianUserId)
      .where(sql`${table.isActive} = true`),
    index('practice_memberships_practice_active_idx')
      .on(table.practiceId)
      .where(sql`${table.isActive} = true`),
  ],
);
```

**Column details:**
| Column | Type | Nullable | Description |
|--------|------|----------|-------------|
| membership_id | UUID PK | No | Primary key, auto-generated |
| practice_id | UUID FK -> practices | No | Parent practice |
| physician_user_id | UUID FK -> users | No | Physician on this practice |
| billing_mode | VARCHAR(30) | No | `'PRACTICE_CONSOLIDATED'` or `'INDIVIDUAL_EARLY_BIRD'` — default `'PRACTICE_CONSOLIDATED'` |
| joined_at | TIMESTAMPTZ | No | When the physician was added |
| removed_at | TIMESTAMPTZ | Yes | When removal was initiated (null = not removed) |
| removal_effective_at | TIMESTAMPTZ | Yes | End of calendar month when removal takes effect |
| is_active | BOOLEAN | No | Default true. False when removed |
| created_at | TIMESTAMPTZ | No | Auto-set to now |

**Indexes (critical):**
1. `practice_memberships_practice_physician_active_idx` — Unique partial index on `(practice_id, physician_user_id)` WHERE `is_active = true`. Prevents duplicate active memberships.
2. `practice_memberships_physician_active_idx` — Unique partial index on `(physician_user_id)` WHERE `is_active = true`. Ensures a physician can belong to at most ONE active practice.
3. `practice_memberships_practice_active_idx` — Regular partial index on `(practice_id)` WHERE `is_active = true`. Speeds up headcount queries.

**billing_mode values:**
- `PRACTICE_CONSOLIDATED`: Physician billed through practice's Stripe subscription at clinic rate ($251.10/month or $2,863/year). Practice's Stripe subscription `quantity` counts these members.
- `INDIVIDUAL_EARLY_BIRD`: Physician retains their own individual early bird subscription ($199/month or $2,388/year). NOT included in practice's Stripe subscription quantity. Still a full practice member, counts toward headcount, and appears in the seat list.
