# Task D17-001: Add discount constants to platform.constants.ts

## What to Build

Add discount percentage constants to the existing file `packages/shared/src/constants/platform.constants.ts`.

**Add the following constants after the existing `DUNNING_CANCELLATION_DAY` constant:**

```typescript
// --- Discount Framework (Batch 2: Pricing Lifecycle) ---

export const DISCOUNT_ANNUAL_PERCENT = 5;
export const DISCOUNT_CLINIC_PERCENT = 10;
export const DISCOUNT_CEILING_PERCENT = 15;

export const BASE_MONTHLY_RATE = 279;
export const EARLY_BIRD_MONTHLY_RATE = 199;
export const EARLY_BIRD_RATE_LOCK_MONTHS = 12;
export const EARLY_BIRD_EXPIRY_WARNING_DAYS = 30;

// Minimum effective rate: 85% of base rate ($237.15). No discount combination
// may produce a rate below this floor.
export const MINIMUM_RATE_FLOOR = BASE_MONTHLY_RATE * (1 - DISCOUNT_CEILING_PERCENT / 100);
```

**Rules these constants enforce (from spec B2-1):**
- Annual billing discount: 5% off base rate
- Clinic tier discount: 10% off base rate
- Maximum combined discount ceiling: 15% (clinic + annual stack to exactly the ceiling)
- Early bird rate ($199/month) is a flat rate — no discounts stack on top of it
- No configuration may produce a monthly rate below 85% of $279 ($237.15)

**Do NOT modify** any existing constants. Only append the new block.
