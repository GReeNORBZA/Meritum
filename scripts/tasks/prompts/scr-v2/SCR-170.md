# Task SCR-170: Normalize specialty restrictions: filter garbage fragments, title-case

## What to Build

Add `normalizeSpecialty` function in `scripts/enrich-hsc-data.ts`:

1. Filter values starting with common fragment prefixes: 'physicians', 'those', 'with', 'working', 'in', 'at', 'for', 'by'.
2. Filter values shorter than 4 characters.
3. Filter values ending with prepositions: 'in', 'at', 'for', 'by', 'with', 'of', 'who', 'that'.
4. Title-case the first character.
5. Apply in `extractSpecialtyRestrictions` after splitting, before dedup.
