# Task SCR-154: Wrap all seed inserts in a database transaction

## What to Build

In `apps/api/src/seed.ts`:

Wrap all insert operations inside `await db.transaction(async (tx) => { ... })`. Replace all `db.insert(...)` calls with `tx.insert(...)` inside the transaction callback.

The idempotency check (line 213-218) should remain OUTSIDE the transaction.

This ensures that if any insert fails, all changes are rolled back atomically.
