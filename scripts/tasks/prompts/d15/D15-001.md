# Task D15-001: Fix annual pricing constant and add early bird annual plan constant

## What to Build

Modify `packages/shared/src/constants/platform.constants.ts` to correct the standard annual pricing and add the new early bird annual plan.

**1. Fix STANDARD_ANNUAL pricing:**

In the `SubscriptionPlanPricing` object, change the `STANDARD_ANNUAL` entry:

```typescript
// BEFORE:
[SubscriptionPlan.STANDARD_ANNUAL]: {
  plan: SubscriptionPlan.STANDARD_ANNUAL,
  amount: '2790.00',
  interval: 'year',
  label: 'Standard Annual',
},

// AFTER:
[SubscriptionPlan.STANDARD_ANNUAL]: {
  plan: SubscriptionPlan.STANDARD_ANNUAL,
  amount: '3181.00',
  interval: 'year',
  label: 'Standard Annual',
},
```

Rationale: $279.00 x 12 x 0.95 = $3,181.32, rounded to $3,181.00 per the rate card. This is a 5% annual discount, not the incorrect ~17% discount the old value represented.

**2. Add EARLY_BIRD_ANNUAL to SubscriptionPlan enum:**

```typescript
// BEFORE:
export const SubscriptionPlan = {
  STANDARD_MONTHLY: 'STANDARD_MONTHLY',
  STANDARD_ANNUAL: 'STANDARD_ANNUAL',
  EARLY_BIRD_MONTHLY: 'EARLY_BIRD_MONTHLY',
} as const;

// AFTER:
export const SubscriptionPlan = {
  STANDARD_MONTHLY: 'STANDARD_MONTHLY',
  STANDARD_ANNUAL: 'STANDARD_ANNUAL',
  EARLY_BIRD_MONTHLY: 'EARLY_BIRD_MONTHLY',
  EARLY_BIRD_ANNUAL: 'EARLY_BIRD_ANNUAL',
} as const;
```

**3. Add EARLY_BIRD_ANNUAL to SubscriptionPlanPricing:**

Add a new entry after the EARLY_BIRD_MONTHLY entry:

```typescript
[SubscriptionPlan.EARLY_BIRD_ANNUAL]: {
  plan: SubscriptionPlan.EARLY_BIRD_ANNUAL,
  amount: '2388.00',
  interval: 'year',
  label: 'Early Bird Annual',
},
```

Rationale: $199 x 12 = $2,388.00. No additional annual discount stacks on early bird pricing.

**4. Update CLAUDE.md line 9:**

In the root `CLAUDE.md` file, change the pricing reference line from:

```
- **Pricing:** $279/month, $2,790/year, $199/month early bird (first 12 months, first 100 physicians)
```

to:

```
- **Pricing:** $279/month, $3,181/year, $199/month early bird, $2,388/year early bird annual (first 12 months, first 100 physicians)
```

**Pricing strategy reference:** $279/month standard, $3,181/year standard annual (5% discount), $199/month early bird, $2,388/year early bird annual.
