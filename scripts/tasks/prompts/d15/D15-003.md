# Task D15-003: Update service layer for early bird annual

## What to Build

Update the platform service and repository layers to support the new `EARLY_BIRD_ANNUAL` plan.

### 1. Update `apps/api/src/domains/platform/platform.service.ts`

**a) Update the `createCheckoutSession()` early bird cap check:**

The existing code only checks the cap for `EARLY_BIRD_MONTHLY`. It must also apply the 100-cap check to `EARLY_BIRD_ANNUAL`.

```typescript
// BEFORE:
if (plan === SubscriptionPlan.EARLY_BIRD_MONTHLY) {
  const earlyBirdCount = await deps.subscriptionRepo.countEarlyBirdSubscriptions();
  if (earlyBirdCount >= EARLY_BIRD_CAP) {
    throw new BusinessRuleError(
      'Early bird plan is sold out',
      { code: 'EARLY_BIRD_SOLD_OUT' },
    );
  }
}

// AFTER:
if (plan === SubscriptionPlan.EARLY_BIRD_MONTHLY || plan === SubscriptionPlan.EARLY_BIRD_ANNUAL) {
  const earlyBirdCount = await deps.subscriptionRepo.countEarlyBirdSubscriptions();
  if (earlyBirdCount >= EARLY_BIRD_CAP) {
    throw new BusinessRuleError(
      'Early bird plan is sold out',
      { code: 'EARLY_BIRD_SOLD_OUT' },
    );
  }
}
```

### 2. Update `apps/api/src/domains/platform/platform.repository.ts`

**Update `countEarlyBirdSubscriptions()` to count BOTH early bird plans:**

The current implementation only counts `EARLY_BIRD_MONTHLY`. It must count both `EARLY_BIRD_MONTHLY` and `EARLY_BIRD_ANNUAL` toward the 100-physician early bird cap.

```typescript
// BEFORE:
async countEarlyBirdSubscriptions(): Promise<number> {
  const rows = await db
    .select()
    .from(subscriptions)
    .where(eq(subscriptions.plan, 'EARLY_BIRD_MONTHLY'));
  return rows.length;
},

// AFTER:
async countEarlyBirdSubscriptions(): Promise<number> {
  const rows = await db
    .select()
    .from(subscriptions)
    .where(
      sql`${subscriptions.plan} IN ('EARLY_BIRD_MONTHLY', 'EARLY_BIRD_ANNUAL')`
    );
  return rows.length;
},
```

Note: The `sql` import from `drizzle-orm` is already present in the file. Alternatively, use `inArray` from drizzle-orm if preferred, but the `sql` template approach is consistent with existing patterns in this file.

**Pricing strategy reference:** $279/month standard, $3,181/year standard annual (5% discount), $199/month early bird, $2,388/year early bird annual. The 100-physician early bird cap applies to both monthly and annual early bird plans combined.

## Critical Security Rules

- The Stripe price ID for `EARLY_BIRD_ANNUAL` must come from the environment variable `STRIPE_PRICE_EARLY_BIRD_ANNUAL`. Never hardcode Stripe price IDs.
- No PHI is sent to Stripe. Only physician name and email.
- Early bird cap must be enforced server-side. The count must include BOTH early bird plan types.
